# MIDIUSB_1x1
MIDI Interface 1 in 1 out based on Arduino MidiUSB and Midi (https://github.com/FortySevenEffects/arduino_midi_library) libraries

Works with iPad and ios native midi drivers (must patch MidiUSB power settings as instructed in the .ino file)
